Meu Primeiro Jogo

Nesse projeto, queria criar uma jogabilidade fluida. Ande e pule apenas. Com o tempo eu quiz mudar muitas coisas no game ele ele ficou uma bagunça ent com medo de ter outro projeto largado eu resolvi ignorar animações, som e um boss para finalizar o game com apenas 8 fases.

Os controles são basicos, setas horizontais e espaço. Uma dica que nao botei no game para aumentar a dificuldade é existem fases nas quais o portal so aparece caso você destrua os canhoes( tocando neles ou usando eles para se autodestruirem

Clique aqui(ou na imagem) e baixe o jogo

Ao baixar o Build3.0.zip você deve arrastar para uma pasta de sua escolha, extrair o arquivo compactado e excecutar o "MeuPrimeiroJogo.exe".
