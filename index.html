My Projects

    As part of my experiments to better understand neural networks and their applications, I started by learning the basics using PyTorch. Click here for more details about this first step.

    This is the second part of my plan to understand neural networks. I implemented a neural network from scratch using C++ without any libraries. The goal was to achieve results similar to the Python project previously mentioned. Click here for more details.

My tasks included: 1) creating a haptic syringe for an anesthesiology simulator, and 2) developing a simulator for studying anesthesiology. More details can be found in the GitHub repository.
